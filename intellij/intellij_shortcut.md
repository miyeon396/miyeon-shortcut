# IntelliJ 단축키

- Cmd + Option + v : 변수 추출
- Cmd + Option + m : 메서드 추출
- Cmd + Option + b : 구현체 바로가기
- Shift + F6 : 변수 한번에 변경
- sout : System.out.println()
- psvm : public static void main(String[] args) {}
- F2 : 오류난 곳으로 바로 이동
- Cmd + , : 설정
- Cmd + ; : 프로젝트 구조
- Cmd + shift + t : test class 생성
- Cmd + e : 과거 history 다나옴
- Cmd + Shift + enter : 바로 다음 줄 시작으로 이동
- Cmd + o : class
